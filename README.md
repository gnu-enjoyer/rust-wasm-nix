# Bevy template for reproducible Wasm builds with Nix

Features:    

* Nix for reproducible builds
* Portable CI/CD
* .devcontainer for GitHub Codespaces

Live preview (slow, debug build):
    https://gnu-enjoyer.github.io/rust-wasm-nix/

Dev env:
    `nix develop`

Build:
    `make build`

Debug:
    `make debug`
